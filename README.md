# 🎮 Lab 404

Bem-vindo ao **Lab 404**!

Este projeto é um RPG 2D desenvolvido em **C++** utilizando **Raylib**.

O objetivo é ensinar conceitos de Arquitetura de Computadores de forma divertida, através de exploração, missões e desafios.

## 🚀 Tecnologias

- C++
- Raylib
- Git
- GitHub

## 📌 Roadmap

- [ ] Personagem
- [ ] NPC Professor
- [ ] Primeiro mapa
- [ ] Sistema de diálogo
- [ ] Primeiro inimigo
- [ ] Sistema de missões
- [ ] HUD
- [ ] Animações

## 📚 Objetivo

Criar um jogo educativo de código aberto que possa evoluir ao longo da faculdade.
